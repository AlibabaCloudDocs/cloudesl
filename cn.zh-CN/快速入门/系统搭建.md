# 系统搭建

用户可根据以下步骤搭建云价签业务系统。

1.  注册阿里云账号。

    官网地址：[https://www.aliyun.com](https://www.aliyun.com)。

    **说明：** 已有阿里云账号可忽略此步骤。

    ![注册阿里云账号](https://static-aliyun-doc.oss-accelerate.aliyuncs.com/assets/img/zh-CN/1305076061/p186190.png)

2.  申请加入公测。

    **说明：** 只有首次使用云价签系统时需要申请，后续使用无需再申请。

    1.  使用阿里云账号登录[云价签控制台](https://cloudesl.console.aliyun.com)。

        系统显示公测申请界面，按照界面提示进入页面申请权限。

        ![公测申请](https://static-aliyun-doc.oss-accelerate.aliyuncs.com/assets/img/zh-CN/1443186061/p186980.png)

    2.  填写信息，单击**submit**完成权限申请。

        ![云价签公测](https://static-aliyun-doc.oss-accelerate.aliyuncs.com/assets/img/zh-CN/8533186061/p186976.png)

    公测申请审批通过后，登录[云价签控制台](https://cloudesl.console.aliyun.com)可以进行云价签相关配置。

3.  创建RAM用户。

    创建RAM用户步骤请参见[权限管理](/cn.zh-CN/用户指南/权限管理.md)。

4.  申请RAM用户AccessKey和AccessSecret。

    **说明：** AccessKey和AccessSecret是您访问阿里云 API 的密钥，具有该账户完全的权限，请您务必妥善保管！不要通过任何方式（e.g. Github）将 AccessKey 公开到外部渠道，以避免被他人利用而造成 安全威胁。强烈建议您遵循阿里云安全最佳实践，使用 RAM用户AccessKey来进行API调用。

    1.  使用阿里云账号或者添加到管理员用户组的RAM用户登录[RAM控制台](https://ram.console.aliyun.com/policies)，将鼠标放至头像处单击**AccessKey管理。**

        ![AccessKey管理](https://static-aliyun-doc.oss-accelerate.aliyuncs.com/assets/img/zh-CN/2490507061/p187344.png)

    2.  单击**开始使用子账户AccessKey**。

        ![开始使用](https://static-aliyun-doc.oss-accelerate.aliyuncs.com/assets/img/zh-CN/0897986061/p187347.png)

    3.  根据实际情况创建用户或者使用已经创建好的用户。

        **说明：** 使用SDK访问云价签系统时，编程访问要勾选。

        ![编程访问](https://static-aliyun-doc.oss-accelerate.aliyuncs.com/assets/img/zh-CN/0897986061/p187349.png)

    4.  在**人员管理** \> **用户**界面，单击要启用的用户，单击**创建AccessKey**。

        **说明：** 请妥善保存AccessKey。

        ![创建AccessKey](https://static-aliyun-doc.oss-accelerate.aliyuncs.com/assets/img/zh-CN/0897986061/p187361.png)

5.  导入商品库。

    使用步骤四创建的AK，导入商品库到云价签系统，商品库导入步骤请参见[t2047327.md\#](/cn.zh-CN/开发指南/导入商品库.md)。

6.  绑定基站。

    1.  新增基站，详细步骤操作请参见[新增基站](/cn.zh-CN/用户指南/价签基站管理.md)。

    2.  激活基站，详细操作步骤请参见[激活基站](/cn.zh-CN/用户指南/价签基站管理.md)。

    3.  将基站接入门店网络。

        -   若基站支持POE供电，通过网线将基站与交换机互联。网线一端接入交换机网口，另一端接入基站WAN/POE口（黄色口）。
        -   若基站不支持POE供电，需要POE供电模块上电后，再通过网线将基站与交换机互联。
            1.  第一根网线将基站WAN/POE口（黄色口）与POE供电模块POE口互联。
            2.  另一根网线将POE供电模块LAN口和交换机网口相连。
        基站电源灯常亮，表示入网正常，可以进行电子价签操作。若电源灯闪烁，表示基站与互联网网络不通，咨询负责现场网络的IT工程师，网络环境是否是动态IP获取环境或者是否有入网限制。

        基站电源灯常见状况如下：

        |闪烁间隔|原因|
        |----|--|
        |常亮|正常工作|
        |50ms|没有IP地址|
        |200ms|到网关没通|
        |1000ms|到公网不通|
        |2000ms|DNS有问题|
        |4000ms|其他原因导致连不上云端|
        |不亮|没通电|

7.  管理价签。

    -   使用云价签系统绑定价签。

        登录[云价签控制台](https://cloudesl.console.aliyun.com)，在**业务管理** \> **绑定管理**界面，选择需要绑定商品的门店，单击**新增绑定**，输入**商品条码**和**价签条码**，单击**提交**完成商品绑定。

        ![绑定商品](https://static-aliyun-doc.oss-accelerate.aliyuncs.com/assets/img/zh-CN/8806316161/p187702.png)

    -   使用PDA设备绑定价签。

        详细操作步骤请参见[门店使用](/cn.zh-CN/快速入门/门店使用.md)。

    -   使用钉钉小程序绑定价签。

        详细操作步骤请参见[钉钉小程序管理](/cn.zh-CN/用户指南/钉钉小程序管理.md)。
